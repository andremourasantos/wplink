Como funciona?

Para criar um link de WhatsApp basta que você digite o número de telefone, com o DDD, e, opcionalmente, você também pode adicionar uma mensagem, após isso, clique em "Obter link" e pronto!

Seu link será criado através da API oficial do WhatsApp e não contará com nenhum tipo de propaganda ou link suspeito.

O número de telefone e mensagem inseridas ficam salvos no seu navegador, localmente, para facilitar o uso da ferramenta.

Desenvolvido por André Moura Santos. Ícones de Phospor Icons e Flat Icon. Feito no 🇧🇷 Brasil.

Sobre o projeto

O objetivo desse projeto é proporcionar uma ferramenta útil e de livre acesso para criar links de WhatsApp sem propaganda, com foco exclusivo no Brasil.

Confira as últimas novidades da ferramenta clicando aqui ou pressione Alt + L!

Todo o código da página é de livre acesso e se encontra no repositório do GitHub de André Moura Santos, com licença MIT.

Doe para o projeto

A ferramenta de Link de WhatsApp é gratuita e sem anúncios ou subscrições, pois é um projeto solo criado sem motivação financeira.

Para auxiliar novos programadores com o seu desenvolvimento, o código da página também está publicamente disponível, comentado e com licença de uso livre.

Você pode incentivar a atualização deste projeto e na criação de mais ferramentas ao doar qualquer quantia!
