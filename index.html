Como funciona?

Para criar um link encurtado do WhatsApp, você precisa apenas do número de telefone. Insira essas informações nos respectivos campos e clique em "Criar meu link". Todos os links criados pela ferramenta adicionarão o DDD do Brasil (55) automaticamente.

Adicionalmente, você pode inserir uma mensagem pré-formatada no seu link, para que ela apareça quando o visualizarem. Para isso, basta preencher o campo "Digite sua mensagem".

Sobre o projeto

Este projeto foi desenvolvido com o intuito de oferecer uma ferramenta de fácil uso e visual agradável para criar links encurtados do WhatsApp.

Sem qualquer tipo de anúncio, pagamento ou assinatura, seu uso é livre para todos. A ferramenta utiliza a API oficial do WhatsApp para criação de links encurtados.

Esse projeto não está afiliado ao WhatsApp ou a Meta Inc.

Saiba mais sobre o projeto, que conta com licença MIT, através do repositório no GitHub de André Moura Santos.
